CSS, SELECTORS, AND COMBINATORS

3 Types of CSS - Cascading Style Sheets

  1. ->Inline

    2. - same line as tag to be styled -> takes highest priority (no code reuse / not recommended)

    <p style="color:green">Hello</p>

  2. ->Embedded/Internal

    3. - written in the head of the html page to be styled (applies code reuse within the same page, average recommendation)

  3. ->External

    4. - written in a seperate file/ external css (100% code reuse / the most recommended)

    *NB :(Priority between Internal and External depends on the position of the linking line and your embedded css style tag...
    if linking line for external comes last, external takes precedence, otherwise internal takes priority)

How does css identify tags to be styled

-> we make use of Selectors

There are 2 types of selectors

  1. Element selectors - selectors based on tag names eg: hr, h1, p, table, tr eg: p{ color: red;}
  2. Named Selectors - Selectors which identifies elements by names (are of 2 types):
    1. -> id (#)- an element is assigned an id, which is used to identify it for styling (only one element may have a unique id) <input type="text" id="fname"> #fname{color: blue;}
    2. -> class (.) - an element is assigned a class name, which is used to identify it for styling (more than one elements may belong to a class) <p class="bida" id="p1">Hi

      <p class="bida">hello</p> <p class="status">hello</p> .bida{letter-spacing:2px;}

*** COMBINING SELECTORS ***

We may combine Selectors as follows:

  1. -> element selector then class (Select a Paragraph with a class called bida)

    2. p.bida{background-color:cyan;}

  2. -> element selector and id (select a div with an id called footer)

    3. div#footer{color:navy;}

*** USING MULTIPLE CLASSES ***

-> elements may be assigned multiple classes for example to set the classes details and status to a div: <div class="details status bida">...</div> (any class that matches' styling will be applied) 

                            .details{ color: navy; }
                            .status { font-weight:bold;}
                            .bida{text-align:center;}
                        

                    
*** GROUPING  ***

                    - Elements sharing the same styles may be grouped to eliminate style redundancies
                    - Example :
                    h1,h2,p{color: blue;}

  1. Child(>)

     -> selects direct children of an element eg:

     div>p{color:blue;}    [picks 4 paragraph] WHICH ONES?

    3. 
                            
    -> selects elements which are grand children/descendants of a given element e.g.
     
                            
     div div{font-weight:bold;} [Picks 3 divs] WHICH ONES?
     

                        
  3. Sibling(+)
    4. 
                            
    -> selects elements which are direct siblings of a given element e.g.
 
                            
 h1+p{color:green;} [selects one p] WHICH ONE?
 

                        
  • General sibling (~)
    • 
                        
     -> selects elements which are  general siblings of a given element e.g.
     
                        
     h1~*{ color: purple; } [selects 3 elements] WHICH ONES?
     
                    

                    
    *** THE ALL SELECTOR (*) ***
    
                    
    -> The all (*) selector selects all elements belonging to the specified element eg: 
     
                    
     body * {
                            font-size:12px;
                        }
                    
    
                
                    

                   
     *** PSEUDO SELECTORS (:) ***
    
                   
     - Helps to select elements upon some action being performed to the element or elements by some qualification eg :first-child
    
                   
     Examples;
                        a:link{color:blue; }
                        a:visited{color:purple;}
                        a:hover{color:pink;}
                        a:active{color:red;}

                        p:first-child{background-color:lime;}
                        p:first-child+(2){background-color:lime;}
                        p:last-child{...}
                        p:first-of-type{...}

                        input:read-only{...}

                        :not(selector)	
                        :not(p)	Selects every element that is not a 
     element
